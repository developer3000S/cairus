# OpenAI Agents SDK

[OpenAI Agents SDK](https://github.com/openai/openai-agents-python) позволяет создавать агентные AI-приложения в лёгком, удобном пакете с минимальным числом абстракций. Это production-ready развитие нашей предыдущей экспериментальной работы с агентами, [Swarm](https://github.com/openai/swarm/tree/main). Agents SDK имеет очень небольшой набор примитивов:

-   **Агенты (Agents)**: LLM с инструкциями и инструментами
-   **Передачи (Handoffs)**: позволяют агентам делегировать задачи другим агентам
-   **Ограничители (Guardrails)**: дают возможность проверять входные данные для агентов.


В сочетании с Python эти примитивы достаточно мощны, чтобы описывать сложные взаимосвязи между инструментами и агентами, и позволяют строить реальные приложения без крутой кривой обучения. Кроме того, SDK содержит встроенную **трассировку (tracing)**, которая помогает визуализировать и отлаживать ваши агентные процессы, а также оценивать их и даже fine-tune модели для вашего приложения.


## Зачем использовать Agents SDK

У SDK есть два базовых принципа проектирования:

1. Достаточно возможностей, чтобы ими было полезно пользоваться, но при этом достаточно мало примитивов, чтобы быстро разобраться.
2. Отлично работает «из коробки», но позволяет тонко настроить, что именно происходит.

Основные возможности SDK:

-   Цикл агента (Agent loop): встроенный цикл, который управляет вызовом инструментов, передачей результатов в LLM и повторением, пока LLM не завершит работу.
-   Python-first: используйте встроенные возможности языка Python для оркестрации и «склеивания» агентов, не нужно осваивать новые абстракции.
-   Хandoffs (передачи): мощный механизм, чтобы координировать работу и делегировать задачи между несколькими агентами.
-   Guardrails (ограничители): выполняйте проверки и валидации входных данных параллельно вашим агентам, прерывая выполнение раньше, если проверки не прошли.
-   Функциональные инструменты: превращайте любую Python-функцию в инструмент с автоматической генерацией схемы и валидацией через Pydantic.
-   Трассировка (Tracing): встроенная трассировка, которая позволяет визуализировать, отлаживать и мониторить ваши рабочие процессы, а также использовать набор инструментов OpenAI для оценки, fine-tuning и дистилляции.


## Установка

```bash
pip install openai-agents
```

## Пример «Hello world»

```python

from cai.sdk.agents import Agent, Runner

agent = Agent(name="Assistant", instructions="Вы — полезный помощник")


result = Runner.run_sync(agent, "Напишите хайку о рекурсии в программировании.")

print(result.final_output)

# Код внутри кода,
# Функции вызывают сами себя,
# Танец бесконечного цикла.

```

(_Если вы запускаете это, убедитесь, что задана переменная окружения `OPENAI_API_KEY`_)

```bash
export OPENAI_API_KEY=sk-...
```
